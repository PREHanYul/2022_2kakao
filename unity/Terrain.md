# Terrain

unity

3D지형물을 생성할 때 사용함.



사용방법

[GameObject] -> [3Dobject] -> [Terrain]

위치,회전,크기 조절 가능.



terrain 옵션

Raise or Lower Terrain

:페인트 브러시툴로 하이트맵을 페인팅함.



Paint Texture

: 표면 텍스처를 적용함.

-> 표면에 재질을 입히는데 사용함. [Create Layer]

Set Height

: 하이트 맵을 특정 값으로 조정함.



Smooth Height

: 하이트맵을 매끄럽게 만들어 터레인 지형을 부드럽게 함.



Stamp Terrain

: 현재 하이트맵 위에 브러시 모양을 스탬핑함.



Paint Trees

: 나무 페인팅을 활성화

나무를 심을 수 있음. 나무를 위한 이미지 파일이 필요함.

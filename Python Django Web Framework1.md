# Python Django Web Framework

###### (1) 기초 

Web Framework란?

어떤 언어를 하나를 가지고 웹 페이지를 만들려고함.

사용자의 요청에 따라서 페이지가 변화하고 이런 것 등등을

그런데 일일이 만드는 것은 힘들다.

웹 페이지 만들때 공통적으로 겹치는 부분들을 미리 해둔 소프트웨어.



###### (2)Django는?

여러 가지 웹 프레임워크 중에서 python을 언어로 한다.

파이선 언어로 하는 것들은 장고 말고도 flask, fastAPI있다.



(3)Port 란?

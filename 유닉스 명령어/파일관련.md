파일/ 디렉토리 관련 명령어

ls - 디렉토리 목록 보기

ls-al- 숨겨진 디렉토리 목록 열기

cd[디렉토리이름] - 해당 디렉토리로 이동하기.

보통 이름 tap눌러서 찾거나 직접 쓰기



pwd - 현재 디렉토리의 위치 보여주기

mkdir - 해당 디렉토리 만들기

rm - 해당파일을 지우기

cp - 파일 복사하기

mv - 이름을 바꾸거나 옮김

touch - 해당 파일을 생성하거나 업데이트

cat - 해당 파일 내용 출력

head - 해당 파일의 첫 10줄만 출력

tail- 해당 파일의 마지막 10줄만 출력

clear - 터미널 창 초기화



------------------------------------



ctrl+C - 실행 강제 마침

!! - 마지막 명령어 반복 실행
